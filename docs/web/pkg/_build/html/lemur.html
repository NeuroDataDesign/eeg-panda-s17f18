    plot(sample_freq)[source]¶

    Constructs a downsampled spark line plot of the time series.

    If there are more than 500 time points, the time series will be down sampled to 500 column variables by windowed averaging. This is done by splitting the time series into 500 equal sized segments in the time domain, then plotting the mean for each segment.

    Parameters:sample_freq (int) – The sampling frequency (how many times sampled per second).
