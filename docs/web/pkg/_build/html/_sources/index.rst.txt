Welcome to Lemur!

Lemur is a library to assist in the aggregate and one-to-one visualization of any set of data points.
This tool was build to help in the visualize multi-modal neuroscience/psychology datasets, but it 
can be used for any set of objects and similarity / dissimilarity function acting on pairs of such 
objects.
