Lemur Data Visualization Package

Red Lemur (Lemur for short) is a visualization package for multimodal datasets.

Quickstart:

pip install redlemur-offbrand

Or, if Python3 is not your default version:

pip3 install redlemur-offbrand

Check out some of the links below for more detailed information about our project!

Docker Container

Web service that utilizes Lemur to generate visualizations for its users.

Quickstart:

docker pull nkumarcc/redlemur

docker run -p 5000:5000 nkumarcc/redlemur

Access the service at localhost:5000! (This link will only work after you have run the service with quickstart instructions)
